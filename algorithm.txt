СПИСОК АЛГОРИТМОВ РАБОТЫ КОДА:

=========================================
PROGRAM1 (Анализ строки и поиск букв)
=========================================
1. ВВОД ДАННЫХ:
   - Считывание всей строки (через getline), включая пробелы.

2. ПРЕДВАРИТЕЛЬНЫЙ ПОДСЧЕТ:
   - Подсчет общего количества пробелов в строке.

3. ПРОВЕРКА НА КОЛИЧЕСТВО СЛОВ:
   - Если пробелов 0 (одно слово), вывод "0" и завершение программы.

4. ПОИСК ПРЕДПОСЛЕДНЕГО СЛОВА:
   - Определение порядкового номера пробела перед предпоследним словом.
   - Поиск индекса этого пробела в строке.

5. ОПРЕДЕЛЕНИЕ ЦЕЛЕВОЙ БУКВЫ:
   - Выбор символа, идущего сразу после найденного пробела (первая буква предпоследнего слова).

6. ПОДСЧЕТ СОВПАДЕНИЙ:
   - Проход по строке и поиск всех слов, чья первая буква совпадает с целевой.

7. ВЫВОД РЕЗУЛЬТАТА:
   - Печать итогового количества найденных слов.


=========================================
PROGRAM2 (Обработка динамического массива)
=========================================
1. ВВОД РАЗМЕРА:
   - Считывание целого числа n для определения размера массива.

2. РАБОТА С ПАМЯТЬЮ:
   - Выделение динамической памяти под массив целых чисел.

3. ЗАПОЛНЕНИЕ И ПЕРВЫЙ ВЫВОД:
   - Заполнение массива числами с клавиатуры.
   - Вывод исходного состояния массива на экран.

4. МОДИФИКАЦИЯ ЭЛЕМЕНТОВ:
   - Циклическая проверка каждого числа:
     а) Если число ЧЕТНОЕ — деление его на 2.
     б) Если число НЕЧЕТНОЕ — прибавление к нему единицы.

5. ВТОРОЙ ВЫВОД:
   - Вывод измененного массива в новую строку.

6. ЗАВЕРШЕНИЕ:
   - Обязательное освобождение динамической памяти (delete []).
   - Завершение работы программы.
